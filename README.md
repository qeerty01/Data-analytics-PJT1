## 1. Project Overview
The Manicure Appointment Booking System is designed to improve the process of scheduling manicure services in beauty salons. Many salons still rely on phone calls or manual messaging for booking appointments, which can result in scheduling conflicts, missed appointments, and inefficient time management.  
This project focuses on analyzing and designing a structured system that allows customers to book appointments easily while enabling salon staff to manage schedules efficiently.

## 3. Problem Statement
Manual appointment booking in manicure salons often leads to double bookings, poor schedule organization, and customer dissatisfaction. Without a structured system, salon staff struggle to manage appointments efficiently. There is a need for a well-analyzed and properly designed system that improves scheduling accuracy and service quality.

---

## 4. Project Objectives
The main objectives of this project are:
- To analyze the current appointment booking process
- To identify functional and non-functional system requirements
- To design a manicure appointment booking system
- To apply System Analysis and Design methodologies
- To improve scheduling efficiency and customer experience

---

## 5. Project Scope
This project focuses on the **analysis and design phase** of the manicure appointment booking system. The scope includes:
- Requirement gathering and analysis
- Identification of system users
- System modeling and documentation
- High-level system design

System implementation and deployment are not included in this project.

---

## 6. Expected Users
The expected users of the system include:
- Salon administrators
- Manicure specialists
- Customers
- System analysts and academic reviewers

---

## 7. System Features (High-Level)
The proposed system will support:
- Appointment booking by customers
- Viewing available time slots
- Appointment management by salon staff
- Organized scheduling and record keeping

## 8. Methodology
